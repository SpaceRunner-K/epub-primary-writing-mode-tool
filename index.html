EPUB の content.opf をブラウザだけで編集

ローカルの EPUB を読み込み、metadata セクションに primary-writing-mode を追加または更新して、修正済み EPUB を再保存します。GitHub Pages のような静的ホストでも動く構成です。

入力 .epub 1ファイル
変更箇所 package document の metadata 内
追加要素 <meta name="primary-writing-mode" content="horizontal-rl"/>
待機中です。EPUB を選んでください。

注: このツールは primary-writing-mode の付与だけを行います。実際の表示結果は EPUB リーダー実装や CSS の writing-mode 指定にも依存します。
